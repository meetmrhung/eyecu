Description of demo:
BeagleBone outputs a test card upon request from the host computer. 
The host computer requests a test card every so often, receives the test card and displays it.

gazeTracker_DebugVideo.c is a modification of the usb_bulk_example application provided with StellarisWare.
usb_dev_bulk.c is a modification of the usb_dev_bulk example included with Starterware. 

Each of these files should be copied into the respective projects to be compiled.
